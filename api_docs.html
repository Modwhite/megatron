Pengenalan

API Megatron membolehkan agensi luar menghantar data ancaman dan menerima skor risiko secara automatik melalui protokol RESTful.

POST /api/v1/forensics/scan

Memulakan imbasan forensik digital ke atas dump fail yang dihantar.

Auth Required { "case_id": "REQ-77421", "source": "Memory_Dump_001.raw", "tools": ["volatility", "regscanner"] }
GET /api/v1/crypto/trace/{wallet_id}

Mendapatkan profil risiko dan sejarah transaksi bagi alamat dompet kripto tertentu.

{ "status": "success", "risk_score": 98, "flag": "High_Risk_Money_Laundering", "last_seen": "2026-03-15T10:30:00Z" }
POST /api/v1/threat/alert

Menghantar notifikasi ancaman siber baru ke Command Center Megatron.

{ "threat_type": "DDoS", "target_ip": "167.99.23.11", "severity": "Critical" }
